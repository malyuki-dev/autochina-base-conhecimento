# 🚗 Autochina CAOA Chery - Base de Conhecimento

Plataforma escalável para gestão de manuais técnicos, vídeos de treinamento e documentação.

## 🛠️ Tech Stack
- **Frontend:** Angular 18 (Signals & Standalone Components)
- **Backend:** NestJS (Node.js Framework)
- **ORM:** Prisma 7 (PostgreSQL)
- **Linguagem:** TypeScript

## 📂 Estrutura do Projeto
- `/autochina-frontend`: Interface do usuário e consumo de APIs.
- `/autochina-backend`: API RESTful e integração com banco de dados.
